**Simulated Case Texts and LLM-Generated Summaries**

Overview
This repository contains an RTF document with the full text of all eleven simulated cases used in our research project
Each case is presented in its entirety, followed by the paragraph summaries generated by the large language models (LLMs) evaluated in this study.

Document Structure
Within the RTF file:
Each simulated case appears first in full.
Immediately following the body of each case, the corresponding paragraph summaries are listed.
Each summary includes the identity of the LLM that generated it. The LLM name appears in parentheses at the end of the corresponding summary.
